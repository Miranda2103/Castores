Lista de pasos para correr su aplicación.
	Clonar repositorio: https://github.com/Miranda2103/Castores.git Rama: Main
	Abrir el proyecto en Visual Studio 2022.
	Ejecutar los scripts que se encuentran en CC.Api\Scripts, creando primeramente la base de datos llamada CASTORES
	En un gitbash instalar las dependencias con el comando npm install
	Inicializar la aplicacion en un gitbash con el comando npm start
	Configurar los proyectos de inicio (CC.Api y CC.Web) para que abra ambos al mismo tiempo
